# 17. Resource requests

ఈ work కి ఒక extra resource కావాలి.
We'll need one extra resource for this work.

Bandwidth సరిపోవట్లేదు, one more person help అవుతుంది.
Bandwidth is short, one more person would really help.

ఇంకొక dev ని onboard చేయొచ్చా?
Can we onboard one more dev?

ఈ scope కి current team చాలదు.
The current team isn't enough for this scope.

ఒక QA resource అదనంగా కావాలి.
I need one additional QA resource.

ఇంత workload single person handle చేయలేడు.
This much workload can't be handled by a single person.

ఆ tool కి license కావాలి.
We'll need a license for that tool.

ఈ access కోసం request raise చేస్తాను.
I'll raise a request for this access.

ఈ environment మాకు provision చేయించగలరా?
Can you get this environment provisioned for us?

ఇది short-term loan resource గా కూడా ok.
A short-term loaned resource works for me too.

ఈ సమయానికి backfill కావాలి.
We need a backfill for this period.

ఇంకొక laptop allocation కావాలి new joiner కి.
We need one more laptop allocation for the new joiner.

ఇది budget approval అడగాలి.
We'll need to ask for budget approval.

ఇది cloud cost మీద impact ఉంటుంది, మీరు ok నా?
This will impact cloud cost, are you ok with it?

Additional headcount request approve అవుతుందా?
Will the additional headcount request get approved?

ఈ skill set ఉన్న వాళ్ళు మన team లో లేరు.
We don't have someone with this skill set on the team.

ఒక SME ని pull in చేయొచ్చా కొంత time?
Can we pull in an SME for some time?

Temporary గా ఒక contractor తీసుకోవాలి.
We'll need to bring in a contractor temporarily.

ఈ infra కోసం ticket raise చేసాను.
I've raised a ticket for this infra.

ఇది hardware request, IT side నుండి process.
This is a hardware request, will go through IT.

ఇది prod access కావాలి, approval pending.
I need prod access, approval is pending.

ఈ DB instance కోసం sizing request పెట్టాలి.
We need to put in a sizing request for this DB instance.

ఈ tool extension కావాలి, lapse అవుతోంది.
We need an extension on this tool, it's about to lapse.

ఇది request pipeline లో ఉంది already.
This request is already in the pipeline.

ఇది leadership నుండి approval అవ్వాలి.
This will need approval from leadership.

Cost-benefit చూపించి pitch చేద్దాం.
Let's pitch it with a cost-benefit angle.

ఇది deny అయితే plan B ఏంటి?
If this gets denied, what's our plan B?

ఈ resource ఎన్ని weeks కావాలి అని clear గా చెప్పండి.
Please be clear on how many weeks you'll need this resource.

ఇది justify చేయాలి numbers తో.
We need to justify this with numbers.

Resource ask ముందుగానే ఇస్తే better.
It's better to put in the resource ask early.
