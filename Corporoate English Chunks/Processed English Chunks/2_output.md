# 2. Asking for updates

ఆ task ఎక్కడ ఉంది?
Where are we on that task?

Quick update ఇస్తావా?
Can you give me a quick update?

దాని status ఏంటి?
What's the status on that?

ఎంత వరకు వచ్చింది?
How far along are we?

ఏదైనా progress ఉందా?
Any progress on this?

Yesterday చెప్పిన దాని మీద update ఏంటి?
Any update on what we discussed yesterday?

Done అయిందా లేక still pending?
Is it done or still pending?

ఇంకా ఎంత time పడుతుంది?
How much more time do you need?

By when expect చేయొచ్చు?
By when can I expect it?

ఏదైనా blocker ఉందా?
Are there any blockers?

Help కావాలా ఎవరి నుండైనా?
Do you need help from anyone?

ఆ ticket close అయిందా?
Did that ticket get closed?

Client నుండి response వచ్చిందా?
Did we hear back from the client?

EOD కల్లా ready అవుతుందా?
Will it be ready by EOD?

Any luck with that issue?
Any luck with that issue?

ఆ piece ఎక్కడ ఉంది ఇప్పుడు?
Where does that piece stand now?

Just checking in — ఏదైనా update ఉందా?
Just checking in — any updates?

ఇవ్వాళ wrap up అవుతుందా?
Will it wrap up today?

ఆ deployment జరిగిందా?
Did the deployment go through?

Testing complete అయిందా?
Is testing complete?

Approval వచ్చిందా?
Did we get the approval?

ఆ call ఎలా జరిగింది?
How did that call go?

Anything pending from your side?
Anything pending from your side?

ఒక్క quick sync చేద్దామా?
Should we do a quick sync?

ETA ఏంటి దానికి?
What's the ETA on that?

Still on track ఉన్నామా?
Are we still on track?

ఏదైనా change ఉందా plan లో?
Any changes in the plan?

Latest status ఒకసారి చెప్పు.
Walk me through the latest status.

ఎక్కడైనా stuck అయ్యావా?
Are you stuck somewhere?

ఒక్క ping వేస్తావా update తో?
Can you drop me a ping with an update?
