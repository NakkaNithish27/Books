# 164. Navigating hierarchy

ఇది నేరుగా skip manager కి తీసుకెళ్ళడం correct కాదు, ముందు manager కి చెప్పాలి.
I shouldn't take this directly to my skip, I need to inform my manager first.

Chain of command follow అవ్వాలి కదా.
We need to follow the chain of command.

Manager loop లో లేకుండా decision తీసుకోవడం awkward అవుతుంది.
Taking a decision without keeping the manager in the loop would feel awkward.

ఈ escalation ముందు manager తో align అవ్వాలి.
Before this escalation, I need to align with my manager.

Director level కి వెళ్ళే ముందు ఒకసారి manager తో check చేస్తా.
I'll check with my manager once before taking it to the director level.

ఈ point manager కి flag చేశాక, తర్వాత leadership కి తీసుకెళ్దాం.
Let's flag this to the manager first, then take it up to leadership.

Senior leader తో direct mail పంపే ముందు manager కి cc పెట్టాలి.
Before sending a direct mail to the senior leader, I should cc my manager.

ఈ skip-level meeting లో manager కూడా ఉండాలి.
My manager should also be in this skip-level meeting.

Hierarchy bypass చేస్తే, manager కి bad అనిపిస్తుంది.
If I bypass the hierarchy, the manager will feel bad about it.

Right channel లో raise చేయాలి, direct గా కాదు.
We need to raise it through the right channel, not directly.

ఈ decision నా level లో కాదు, పైకి push చేయాలి.
This decision isn't at my level, I need to push it up.

Manager approval లేకుండా move అవలేను.
I can't move ahead without my manager's approval.

ఇది leadership call, నేను decide చేయలేను.
This is a leadership call, I can't decide on it.

మీ level లో ఇది sign-off అవ్వాలి.
This needs sign-off at your level.

Senior leader ముందు మాట్లాడేటప్పుడు careful గా ఉండాలి.
I need to be careful when speaking in front of a senior leader.

Cross-team escalation manager రూట్ తోనే వెళ్ళాలి.
Cross-team escalations should go through the manager route only.

Skip తో నేను directly message పెట్టను, manager కి చెప్పి చేస్తా.
I won't directly message my skip, I'll do it through my manager.

ఈ proposal ముందు team lead approval అవ్వాలి, తర్వాత పైకి.
This proposal needs the team lead's approval first, then it can go up.

ఈ meeting కి నేను అడిగితే awkward, manager invite చేయించాలి.
It would be awkward if I asked to be in this meeting, my manager should get me invited.

Director ఏం expect చేస్తున్నారో ముందు manager నుండి తెలుసుకుంటా.
I'll first check with my manager on what the director is expecting.

ఈ ask నా scope లో లేదు, పైకి తీసుకెళ్ళాలి.
This ask isn't in my scope, it needs to go up.

Manager కి blindside అవ్వకూడదు ఈ విషయం లో.
The manager shouldn't be blindsided on this.

Hierarchy respect చేయాలి, otherwise relationship damage అవుతుంది.
We need to respect the hierarchy, otherwise the relationship gets damaged.

పైకి తీసుకెళ్ళే ముందు proper context ఇవ్వాలి.
Before escalating it up, we need to give proper context.

Senior లతో interaction లో politeness maintain చేయాలి.
We need to maintain politeness when interacting with seniors.

ఈ formal communication leadership కి అయితే, language careful గా ఉండాలి.
If this formal communication is going to leadership, the language needs to be careful.

ఈ topic skip-level లో raise చేస్తే, manager కి ముందుగానే heads up ఇస్తా.
If I'm raising this at the skip-level, I'll give my manager a heads up first.

ఈ ask director కి directly కాకుండా, manager through వెళ్ళాలి.
This ask should go to the director via my manager, not directly.

పై వాళ్ళతో interaction లో short and crisp గా మాట్లాడాలి.
Interactions with senior folks should be short and crisp.

ఈ meeting లో senior leader ఉన్నారు, basics మాత్రమే cover చేస్తా.
There's a senior leader in this meeting, I'll just cover the basics.

ఈ feedback skip కి ఇవ్వాలి అంటే proper forum లో ఇస్తా.
If this feedback has to go to my skip, I'll give it in the right forum.

I can't commit on this, my manager needs to confirm.

ఈ scope పైకి push చేస్తే clarity వస్తుంది.
If we push this scope up, we'll get clarity.

I can't commit to the dates asked at the higher level, I need to align with my manager.

Hierarchy ఎలా ఉన్నా, respect తో deal చేయాలి.
Whatever the hierarchy is, we need to deal with it respectfully.
