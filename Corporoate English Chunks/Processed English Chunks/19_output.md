# 19. Rescheduling

ఈ meeting reschedule చేయొచ్చా?
Can we reschedule this meeting?

ఈ time work అవ్వదు, push చేద్దామా?
This time doesn't work, can we push it?

Sorry, ఇంకొక conflict వచ్చింది.
Sorry, another conflict came up.

ఇది tomorrow same time కి move చేయొచ్చా?
Can we move it to the same time tomorrow?

ఒక గంట late చేద్దాం.
Let's push it by an hour.

ఈ slot లో నేను available కాదు.
I'm not available at this slot.

ఇంకొక slot suggest చేస్తారా?
Can you suggest another slot?

ఇది next week కి postpone చేయాలి.
We need to postpone this to next week.

Last minute change సారీ.
Sorry for the last-minute change.

ఇది 30 mins ముందుకి pull చేయొచ్చా?
Can we pull this 30 mins earlier?

ఇది after lunch కి move చేస్తే better.
It's better to move this to after lunch.

Calendar update చేస్తాను new time తో.
I'll update the calendar with the new time.

ఈ recurring meeting ని pause చేద్దాం కొంత time.
Let's pause this recurring meeting for some time.

ఈ session cancel చేసి తర్వాత fresh గా పెడదాం.
Let's cancel this session and set it up fresh later.

ఇది అత్యవసరంగా reschedule చేయాలి.
We need to reschedule this urgently.

మీకు ఏ time suit అవుతుందో చెప్పండి.
Let me know what time suits you.

ఇది Friday కి move చేయొచ్చా?
Can we move it to Friday?

ఈ meeting half hour లో అయిపోదు, longer slot పెట్టుకుందాం.
This meeting won't wrap in half an hour, let's block a longer slot.

ఈ time అందరూ available కాదు, shift చేద్దాం.
Not everyone's available at this time, let's shift.

Stakeholder availability ప్రకారం adjust చేద్దాం.
Let's adjust based on stakeholder availability.

ఇది 4 PM నుండి 5 PM కి move అవుతోంది.
This is being moved from 4 PM to 5 PM.

ఈ rescheduling కోసం invite update వెళ్తుంది.
An updated invite will go out for this rescheduling.

ఈ meeting decline చేసాను, please reschedule.
I had to decline this meeting, please reschedule.

ఈ slot కోసం నన్ను represent చేయడానికి ఎవరినైనా అడుగుతాను.
I'll ask someone to represent me at this slot.

ఇది short notice కాబట్టి కొంత మంది join అవ్వకపోవచ్చు.
Since this is short notice, some may not be able to join.

ఈ meeting ని tomorrow EOD కి push చేయాలి.
We need to push this meeting to tomorrow EOD.

ఇది schedule conflict వల్ల reschedule.
Rescheduling due to a schedule conflict.

Sorry, మాకు ఆ time లో ఇంకొక priority call వచ్చింది.
Sorry, we had another priority call at that time.

మీకు ok అయితే 30 mins late start చేద్దాం.
If you're ok, let's start 30 mins late.

ఇది ఇక postpone చేయకుండా ఈసారి జరగాలి.
Let's not postpone this again, it has to happen this time.
