# 77. Accountability discussions

ఒక straight talk కావాలి దీని మీద.
We need a straight conversation on this.

Commitment miss అయ్యింది last week.
The commitment was missed last week.

ఇది ఎందుకు జరిగింది?
What happened here?

ఎవరి side నుండి miss అయ్యిందో clarity కావాలి.
I need clarity on whose side this got missed.

Blame game కాదు, just understand అవుతాను.
This isn't about blame, I just want to understand.

ఇది repeatedly జరుగుతోంది ఇప్పుడు.
This is happening repeatedly now.

ఒకసారి అయితే ok, కానీ pattern అవుతోంది.
Once is okay, but it's becoming a pattern.

అది handle చేయాలి seriously.
We need to handle this seriously.

మీ side నుండి commit చేసిన date miss అయ్యింది.
The date you committed to was missed.

ఈ సారి ఏం different గా చేస్తారు?
What will you do differently this time?

ఒక concrete plan ఇవ్వండి.
Give me a concrete plan.

Just promise కాదు, action కావాలి.
Not just a promise, I need action.

ఇది నా mistake, accept చేస్తున్నాను.
This was my mistake, I'll own it.

Next time ఇది repeat అవ్వకుండా చూస్తాను.
I'll make sure this doesn't repeat next time.

ఇది ownership కి direct related.
This ties directly to ownership.

Commitments తీసుకుంటే deliver చేయాలి.
If we make commitments, we have to deliver.

అవ్వకపోతే ముందుగానే flag చేయాలి.
If not, we should flag it early.

Silent గా slip అవ్వడం acceptable కాదు.
Silent slips aren't acceptable.

Status updates regular గా రావాలి నాకు.
I need regular status updates.

Surprise లేకుండా ఉండాలి end లో.
There shouldn't be surprises at the end.

ఈ task ఎవరి accountability?
Whose accountability is this task?

అది clear గా document చేద్దాం.
Let's document that clearly.

Next time miss అయితే escalation అవుతుంది.
If it's missed next time, this will escalate.

అది honest గా చెప్తున్నాను.
I'm being honest about it.

ఇది trust issue అవ్వకూడదు.
I don't want this to become a trust issue.

Accountability culture build అవ్వాలి team లో.
We need to build an accountability culture in the team.

మీ సహకారం ఉంటే ఇది easy.
With your cooperation, this is easy.
