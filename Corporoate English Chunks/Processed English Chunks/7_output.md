# 7. Prioritization

ఇందులో ఏది ముందు చేయాలి?
Which one should I do first?

Priority order ఏంటి?
What's the priority order?

ఇది P1 యా P2 యా?
Is this P1 or P2?

ఇది top priority గా treat చేయాలా?
Should I treat this as top priority?

ఇంతలో ఏది drop చేయొచ్చు?
Out of these, which can I drop?

ఇంత plate లో ఉంది, ఏది park చేయొచ్చు చెప్పు.
My plate's full — tell me what I can park.

ఇది can wait అనుకుంటున్నాను.
I think this one can wait.

ఇది urgent గా lift చేయాలి.
I need to pick this up urgently.

ఈ task ముందు finish చేస్తే rest unblock అవుతుంది.
If I finish this task first, the rest will get unblocked.

ఇది back burner కి పెట్టొచ్చు.
We can put this on the back burner.

ఇది ఇవ్వాళ touch చేయట్లేదు.
I'm not touching this today.

ఇది tomorrow take చేస్తాను.
I'll pick this up tomorrow.

ఏది ముందు, ఏది తర్వాత చెప్పు.
Tell me what comes first and what comes next.

ఈ rest of items deprioritize చేస్తాను.
I'm going to deprioritize the rest of these items.

ఇది must-do today.
This is a must-do today.

ఇంత ఒకేసారి possible కాదు, ఏది important చెప్పు.
All of this at once isn't possible — tell me which one is important.

Bandwidth limit ఉంది, తేల్చుకోవాలి.
I have limited bandwidth, I need to choose.

ఇది high impact ఉంది, ముందు చేస్తాను.
This one's high impact, I'll do it first.

Low hanging fruit ముందు pick up చేద్దాం.
Let's pick the low hanging fruit first.

ఇది blocker కాబట్టి top of the list.
Since this is a blocker, it's top of the list.

ఏది critical ఏది nice-to-have చెప్పు.
Tell me what's critical and what's nice-to-have.

ఇది ఇప్పటికి postpone చేద్దాం.
Let's postpone this for now.

ఇది next sprint కి push చేద్దాం.
Let's push this to next sprint.

Quick wins ముందు knock out చేద్దాం.
Let's knock out the quick wins first.

ఇంకా ఒక new request వచ్చింది, priorities re-align చేయాలి.
Got a new request — I need to re-align my priorities.

ఇంత conflict గా ఉంది, you tell me ఏది ముందు.
Things are conflicting — you tell me what comes first.

Honestly, all P1 అని చెప్తే manage అవ్వదు.
Honestly, if everything is P1, nothing is P1.

ఇది parking lot కి వెళ్తుంది ఇప్పటికి.
This goes to the parking lot for now.

ఇది leadership తో align చేయాలి priority మీద.
We need to align with leadership on the priority.

ఇంకొక్క pair of eyes కావాలి priorities review కి.
I need another pair of eyes to review priorities.
